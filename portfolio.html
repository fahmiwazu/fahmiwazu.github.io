Real-time Monitoring Power Meter via Modbus with Web Dashboard (December 2018)
by Bayu Eko Saputro - Sutriyono

In this project, we built a remote monitoring system to help building owners monitor power usage in real-time. We use a power meter to take power consumption data and then read it using the Modbus protocol. Then from reading the data sent to the web dashboard via the GPRS network.

Restful Web Server Smart Parking System (April 2018)
by Bayu Eko Saputro

In the near future, most of smart device (things) will be build smart-connected-network that serve service to user via internet network, it's called internet of things (IoT). Restful Web Server is one of technology that enabled user to communicate with smart device. The Web Server can store data sensor from smart device and perform so many features like visualisation and analytics.

This is my very first Restful Web Server that i built using laravel framework. Full source code available on my github. Demo : http://smartparkingv1.herokuapp.com

IoT Arduino RESTFUL Web Server
SMALECTER (PKM KC - Juni 2017)
by Soedirman Moeda Team UNSOED

Smalecter is acronym for "Smart Electrical Terminal", iot based portable device like electrical terminal for monitoring current, voltage and power usage and also controlling the activity of user's electronic device that connected to it's socket. Smalecter has two operation mode, near and far. For near mode, user can access web server smalecter using local network via wifi. For far mode, user can access the device using ionic android app that connected to cloud MQTT broker.


IoT MQTT RESTFUL Web Server Ionic
ιota-emc (UNSOED FAIR - November 2017)
by Bayu Eko Saputro, Supervised by Azis Wisnu Widhi N.

ιota is a concept developed by LJK Electrical Engineering Faculty of Engineering Unsoed in build a network of household appliances to facilitate user control and monitoring. ιota is built using the concept of Internet of Things (IoT).

ι-emc is a device used for monitoring and controlling electronic equipment. Control on ι-emc can be done manually or automatically.ι-emc monitor the condition of electronic equipment (active/on or off) as well as monitoring energy use. By monitoring the use of energy then anomaly energy use can be detected. Energy use can be caused by equipment failure. For example, a dirty AC unit will result in increased energy consumption. One of ι-emc's ability is to activate equipment connected to it based on time.
